
# 🎯 Quiz Platform

An interactive and responsive Quiz Platform built using **HTML, CSS, and JavaScript**.
This project allows users to attempt multiple-choice questions, track their score, and enjoy a smooth user experience.

---

## 🚀 Features

* 📝 Multiple-choice quiz questions
* ⏱️ Real-time score tracking
* 🎨 Clean and responsive UI
* 🔄 Dynamic question loading using JavaScript
* 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

* **HTML** – Structure of the application
* **CSS** – Styling and layout
* **JavaScript** – Logic and interactivity
